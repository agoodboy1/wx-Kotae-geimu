### 使用工具
- 微信小程序默认的开发工具
- webstorm


### 需要
- weUI
- WePY
- iconfont

### 运行方法
```
npm install wepy-cli -g
npm install 
wepy build --watch
```
需要自己提供一个已经备案过的https的网站并且改相关接口。
分享功能需要自己准备appid和secret （直接去小程序后台获取即可）
